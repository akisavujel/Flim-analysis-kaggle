## 📌 Overview

A comprehensive **Exploratory Data Analysis** project on a Bollywood movies dataset. The goal is to understand movie budgets, box office performance both in India and overseas, and general movie outcomes through rigorous data cleaning and visualization.

---

## 🎯 Objectives
```
01  Clean and prepare the Bollywood movies dataset
02  Analyze budget distribution and identify outliers
03  Study India and overseas box office performance
04  Understand movie verdict distribution
05  Generate meaningful insights using visualizations
```

## 🔄 Analysis Pipeline
```
📥 Raw Data
   │
   ▼
🧹 Data Cleaning ──── Handle duplicates, fix data types, remove outliers
   │
   ▼
🔍 Exploration ──────  Summary statistics, distributions, value counts
   │
   ▼
📊 Visualization ────  Bar, histogram, scatter, box plots
   │
   ▼
💡 Insights ─────────  Business conclusions and pattern discovery
```

---

## 💡 Key Insights

| # | Insight |
|---|---------|
| 01 | 💰 **Budget is right skewed** — most Bollywood movies are made with low to moderate budgets |
| 02 | 🌍 **Overseas is concentrated** — only a few films dominate international earnings |
| 03 | 🏆 **Blockbuster is rare** — Flop is the most common verdict in Bollywood |
| 04 | 📊 **Net ≈ Gross** — India Net and India Gross are nearly perfectly correlated (r ≈ 0.99) |
| 05 | 🎬 **Two-tier industry** — massive budget gap between blockbusters and average films |
