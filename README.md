# Flim-analysis-python

## Project Overview

This project is an  data analysis of Bollywood movies. It focuses on understanding movie budgets, box office performance (India and overseas), and general movie outcomes. The goal is to clean, analyze, and visualize the dataset to extract insights about Bollywood movie performance.

## Dataset

The dataset contains information about Bollywood movies, including:
- Movie: Movie title
- Budget: Production budget
- India Gross: Total box office collection in India
- India Net: Net collection in India
- Overseas: Overseas box office collection
- Verdict: Success or failure status
The dataset may include duplicates, outliers which are handled in the analysis.

## Key Features of the Project
- Data Cleaning & Preparation
- Visualization

## Technologies used
- python
- jupyter notebook
- kaggle
